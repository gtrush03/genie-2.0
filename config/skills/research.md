# Research Wishes

For "research X" / "find out about Y" / "who is Z" wishes:

## Approach

1. `WebSearch` for recent, authoritative sources
2. `WebFetch` the top 2-4 hits to get real text
3. For people: also use the Playwright MCP to search LinkedIn while logged in — you'll get way more signal than public search
4. Cross-reference multiple sources. Don't trust a single result.
5. Cite your sources in the final Telegram report with URLs

## Quality bar

- Don't just summarize the first Google result. Synthesize across multiple sources.
- For people: LinkedIn profile + recent news + social media presence
- For companies: what they do, recent funding/news, key people, market position
- For topics: explain clearly, include relevant data points and dates
- Always include source URLs so George can dig deeper if needed
